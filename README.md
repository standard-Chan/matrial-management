## Material Management Project

구매 요청부터 발주, 입고, 송장 처리, 지급까지의 전 과정을 관리하는 기업용 구매/회계 통합 시스템입니다.

---
## 구현

### 요청 처리
- GET, POST : SSR 방식으로 HTML 페이지를 렌더링한 후 반환합니다.
- PUT, DELETE : REST API 요청으로 데이터를 서버에 전달합니다.
